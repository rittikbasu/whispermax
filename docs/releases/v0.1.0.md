## whispermax v0.1.0

i wanted a fast local dictation app i could trust, so i built whispermax.

- fully local dictation with whisper large v3 turbo
- compact floating recorder ui
- fast global hotkey workflow
- direct insertion across many native apps and browser text boxes
- honest fallback to copied to clipboard when insertion can’t be trusted
- searchable transcript history
- dictionary for custom names, products, and phrases
- simple onboarding for model, permissions, and hotkey setup

notes:
- this release is local-first and built around one model
- the app is currently not notarized, so macos may ask you to open it manually the first time
